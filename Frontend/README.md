# RFO Frontend Project

A Umi + React 18 + JavaScript + Ant Design 5 + MobX 6 + Axios + less frontend
project.

## Technology Stack

- **Framework**: Umi 4, React 18
- **Language**: JavaScript (ES2022+)
- **UI**: Ant Design 5, @ant-design/icons
- **State**: MobX 6 (`mobx`, `mobx-react-lite`)
- **Networking**: Axios
- **Styles**: less

## Project Structure

```text
src/
├── apis/                    # API functions
├── components/              # Reusable UI components
├── layouts/                 # Layout components
│   ├── MainLayout.jsx
│   └── MainLayout.module.css
├── pages/
│   ├── index.jsx
│   ├── login.jsx
│   ├── setting.jsx
│   └── subject.jsx
├── requests/                # Axios instance/interceptors
├── stores/                  # MobX stores
├── utils/                   # Utilities (optional)
└── .umi/                    # Generated by Umi
```

## Getting Started

### Prerequisites

- Node.js 20 LTS
- pnpm or npm

This project includes a `.nvmrc` file to pin the frontend runtime to Node 20.
If you use a Node version manager, it can read that file and switch automatically
when you enter the `Frontend` directory.

### Install

```bash
# using pnpm
pnpm i
# or npm
npm i
```

### Run dev server

```bash
npm run dev
# or
pnpm dev
```

Build:

```bash
npm run build
# or
pnpm build
```
